# Prediccions d'oracle

> Decisions que oracle ha aprovat amb la seva predicció de com envelliran.
> Oracle revisa periòdicament si les prediccions s'han complert.

---

## Format

```
### [Data] — [Títol de la decisió]
**Decisió aprovada**: [resum]
**Predicció**: [com crec que envelirà — quin problema pot aparèixer i quan]
**Revisió**: [buit fins que es revisi]
```
